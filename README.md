# Lamparas

Proyecto que controla un arduino desde la web.

## ¿Qué se necesita?

Raspeberry Pi 3 B  
(Link a la tablita de los pines) https://pinout.xyz/pinout/  

## Instalación

```bash
yarn
```

## Desarrollo

```bash
yarn start
```

Esto crea un servidor local y se puede ver la página en `localhost:8080`
